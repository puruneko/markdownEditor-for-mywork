## Step1：このDnDが本当にHTML5 DnDか確認すること

次をコード上で確認する：

- dragstart を addEventListener で監視しているか
    
- DataTransfer を使用しているか
    
- drop イベントで処理しているか
    

もし pointer/mousemove ベースなら  
これはHTML5 DnDではない。

---

## Step2：drop成立条件を確認すること

HTML5 DnDの場合、必ず以下が必要：

`dragover で preventDefault()`

これが無ければ drop は成立しない。

---

## Step3：手動操作時に本当に dragstart が発火しているかログを出すこと

`document.addEventListener('dragstart', () => console.log('dragstart fired'))`

これが出ないならHTML5ではない。

---

## Step4：HTML5でない場合は mouse API が唯一正しいE2E方法

その場合 locator.dragTo を使ってはならない。

DnDは pointer trajectory として再現する。

---

## Step5：HTML5 DnDに依存しているなら実装を修正すること

テストを変えるのではなく、  
HTML5仕様に準拠させる。
