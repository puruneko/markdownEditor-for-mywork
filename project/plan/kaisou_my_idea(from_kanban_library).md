

markdownエディタでのタスク管理プロジェクトを同時並行で進めており、このライブラリの汎用性を保ったままmarkdownタスクとの親和性をあげるため、以下の概念の導入と実装を計画します。


# 階層構想の概念の導入

- カードに階層構造の概念を導入する。単なるプロパティではなく、標準装備とする
- セクションはheading要素。
- セクションとは別に、セクション内のまとまりを表す「ユニット」を導入
    - カードをまとめるステータス概念の無いグループ
    - ネストを許容する
- カードグループを導入
    - カードをまとめ、サブグループとして配下に置く
    - 自身もカード。ステータスの概念あり
    - カードグループ表示出ないときは、普通のカードとしてkanbanに表示される
    - ネストを許容する

- グループで「階層」を追加（セクション関連は削除）
    - 階層が選ばれると、コンボボックスの右に「headingレベル」（数値）と、「ユニット表示」（チェック）が表示される。
    - headingレベルはどのレベルでグループ化するかの指定
    - ユニット表示は、headingの下層にユニットグルーピングをするかの指定。ユニットグルーピングは、利用者からの見た目はheadingグルーピングと同じで階層が深くなるだけ。ユニットグルーピングはheadingグルーピングの直下の階層として表示されるため、ユニットがH1に属している場合とH3に属している場合では階層の表示位置が異なる（ユニットの階層位置は揃えない）
    - ユニット表示のユニット名は、heading名より少し小さめに表示する（ユニットであることを見た目で仄めかす）


markdownとの対応。カードはタスクリスト（- [ ]）単位で生成。

```
# セクションH1
## セクションH2
### セクションH3

- ユニット
    - [ ] タスクA
        - [ ] サブa
        - [ ] サブb
            - [ ] サブサブw
            - [ ] サブサブx
    - [ ] タスクB
        - [ ] サブc
        - [ ] サブd
            - [ ] サブサブy
            - [ ] サブサブz

```

# カードグループ表示機能の導入

上記markdownとの対応を例に、カードグループ表示機能の説明を行う。

```
カードグループ（カードグループ表示OFF＝通常時）
------------------
|   ...セクションH3|
|タスクA           |
-------------------
```

- １行目は所属セクション名・ユニット名を階層順に全階層表示（グループ表示がセクションの場合は、グルーピングされていない階層を表示：グループレベル1ならH2から表示）。表示しきれない場合は、右詰め表示。参考情報なので、半分くらいの大きさで目立たないように表示。セクション・ユニットに属していない場合、表示する階層が無い場合、行自体を非表示。


```
カードグループ（最上位）（カードグループ表示ON）
------------------
|    ...セクションH3|
|タスクA          ■|
|[status]サブa     |
|[status]サブb    ■|
-------------------
```

- statusはタグアイコンのようなラウンド長方形。
- ■は展開/非展開ボタン（■は図の便宜上使っているだけで四角アイコンが欲しいわけではない）。カードにサブカードがあれば表示され、新たなカードグループとして展開する。上記例のタスクAの方を押すと、サブaとサブbが新たなカードグループとして表示される。サブbの方を押すと、サブbのみカードグループとして表示される。展開されているカードは展開されてるとわかるように装飾する（暫定で下線を引く）
- ■ボタンは展開/非展開で別アイコンとし、一目でこのカードが展開/非展開かどうかわかるようにする。展開されている状態でカード名を押すと展開先のカードグループにフォーカスする（スクロールしないと画面に見えない場合はそこまでスクロール）。配置はタイトルと同じ行の右端。フリーのSVGなどでボタンの意味が分かりやすくする。

```
カードグループ（親あり）
--------------------
|    ...セクションH3|
|(タスクA)         ■|
|サブb　        　  |
|[status]サブサブw  |
|[status]サブサブx  |
--------------------
```

- ２行目には自分の属するカードグループが表示される。この文字を押すと親カードグループがフォーカスされる。
- ■は×ボタンで、押すと非展開状態となり、このカードグループ表示は消え、親カードグループがフォーカスされる（■は図の便宜上使っているだけで四角アイコンが欲しいわけではない）
- 枠線を「展開されたカードグループ」とわかるようにする（暫定で、枠線は通常より２倍太い間隔の広い点線）

※フォーカスされたカードグループは、枠線が通常より３倍太い状態となりフォーカスされ、フォーカスが外れると元に戻る。



グループ選択コンボボックスの右隣に「カードグループ」のチェックボックスを作り、チェックの場合は上記カードグループ表示を有効にする。また、「全展開」「全集約」(?)ボタンをその右に作成する。
